# 🚀 SIMPLE START GUIDE

## Just 3 Steps to Get Running!

---

## Step 1️⃣: Navigate to Project Folder

Open File Explorer and go to:
```
c:\Users\AKHIL\Desktop\resume\ai-resume-hackathon\
```

---

## Step 2️⃣: Double-Click This File

Find and double-click:
```
START-ALL.bat
```

You'll see TWO command windows open:
- ✅ One says "Backend Server" 
- ✅ One says "Frontend Server"

**IMPORTANT: Keep both windows open!**

---

## Step 3️⃣: Wait & Open Browser

1. **Wait 15 seconds** (servers need time to start)
2. Open your browser (Chrome recommended)
3. Go to: **http://localhost:5173**

---

## ✅ You Should See:

- Purple/blue gradient background
- "AI Resume Builder" at the top
- "Create professional, ATS-optimized resumes in minutes"
- Form fields to enter your information

---

## 🎉 That's It! You're Ready!

Now you can:
- Enter your information
- Use AI to enhance content
- Get ATS score
- Try Job Match feature
- Export PDF

---

## ⚠️ If It Doesn't Work:

1. Make sure BOTH command windows are still open
2. Check for error messages in the windows
3. Wait a bit longer (sometimes takes 20-30 seconds)
4. Try refreshing the browser (F5)
5. Read TROUBLESHOOTING.md for detailed help

---

## 🛑 To Stop the Servers:

Close both command windows (the ones that opened)

---

## 🔄 To Restart:

Just double-click **START-ALL.bat** again!

---

**Easy, right? Now go build some amazing resumes! 🎉**
