## 🐛 Troubleshooting

### If Backend Doesn't Start
- Check if port 5000 is in use
- Restart the terminal
- Run `npm install` again in backend folder

### If Frontend Doesn't Connect
- Verify backend is running
- Check .env file has correct API URL
- Clear browser cache

### If PDF Export Fails
- Ensure name and email are filled
- Try different browser (Chrome recommended)
- Check browser console for errors

### If Demo Laptop Fails
- Have backup video ready
- Show screenshots
- Walk through features verbally
- Use your personal laptop as backup

## 🎬 After the Demo

### Q&A Preparation
Common questions and answers:

**Q: How does the AI work?**
A: "We use custom NLP algorithms that analyze text patterns, replace weak verbs with action verbs, and add quantifiable metrics. It's rule-based, not machine learning, which makes it fast and reliable."

**Q: What makes your ATS scoring accurate?**
A: "We researched how real ATS systems work and implemented the same criteria: contact info, keywords, formatting, experience quality. Our algorithm mirrors industry-standard ATS evaluation."

**Q: How do you plan to monetize?**
A: "Freemium model: basic features free, premium templates and advanced AI features paid. Also B2B sales to universities and recruitment agencies."

**Q: What's your competitive advantage?**
A: "Real-time ATS scoring and Job Match AI are unique features. Plus, we're faster and more user-friendly than competitors."

**Q: Can you scale this?**
A: "Absolutely. The architecture is modular and scalable. We can easily add more templates, support more job roles, and integrate with job boards."

**Q: What about data privacy?**
A: "All processing happens on our servers, no data is sent to third parties. Users can delete their data anytime. We're GDPR compliant."
