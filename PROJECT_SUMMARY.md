## 📊 How AI Works

### Content Enhancement
1. Detect weak verbs → Replace with action verbs
2. Identify passive voice → Convert to active
3. Check for metrics → Add quantifiable results
4. Polish grammar → Ensure professional tone

### ATS Scoring
1. Analyze contact info (15 pts)
2. Check professional summary (10 pts)
3. Evaluate skills section (20 pts)
4. Review experience (25 pts)
5. Verify education (10 pts)
6. Calculate keyword density (10 pts)
7. Assess formatting (10 pts)

### Job Matching
1. Normalize job title
2. Search skills database
3. Filter existing skills
4. Return top recommendations
5. Provide keywords and tips
