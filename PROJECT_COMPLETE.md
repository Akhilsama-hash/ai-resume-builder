## 🎯 How to Use This Project

### For Hackathon Demo (5 minutes)

#### Step 1: Installation (One-time, 5 minutes)
```bash
# Double-click:
install.bat

# This installs all dependencies for both backend and frontend
```

#### Step 2: Start Application (2 minutes)
```bash
# Terminal 1 - Start Backend:
start-backend.bat
# Backend runs on http://localhost:5000

# Terminal 2 - Start Frontend:
start-frontend.bat
# Frontend runs on http://localhost:5173
```

#### Step 3: Open in Browser
```
Navigate to: http://localhost:5173
```

#### Step 4: Demo Flow (5 minutes)
1. **Personal Info** (30 sec)
   - Enter name, email, phone, location
   - Show live preview updating

2. **AI Summary** (30 sec)
   - Click "AI Generate"
   - Show professional summary created

3. **Skills** (20 sec)
   - Add 2-3 skills manually

4. **AI Job Match** ⭐ (1 min)
   - Click "AI Job Match" button
   - Enter "Frontend Developer"
   - Show AI suggestions
   - Select and add skills

5. **Experience with AI** (1 min)
   - Add work experience
   - Type: "worked on web projects"
   - Click AI enhance (⚡)
   - Show transformation

6. **ATS Score** ⭐ (1 min)
   - Point to score card
   - Show score (should be 70-90)
   - Explain real-time feedback

7. **Templates** (30 sec)
   - Switch between Modern, Classic, Creative

8. **Export** (30 sec)
   - Click "Export PDF"
   - Show downloaded resume
